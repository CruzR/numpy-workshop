Vorbereitung

Für die Teilnahme am Workshop benötigst du:

  • einen Laptop mit GNU/Linux
  • etwas Interesse
  • Kenntnisse im grundlegenden Umgang mit der Kommandozeile (zur Installation der benötigten Python-Module)

Nicht notwendig, aber sicher hilfreich, sind dagegen bestehende Programmierkenntnisse.

Installation

Installation der benötigten Distributions-Pakete

Der erste Teil der Installation wird je nach verwendeter Linux-Distribution ein wenig unterschiedlich aussehen. Danach werden wir eine sogenannte Virtualenv, also eine virtuelle Python-Umgebung, verwenden, um die benötigten Python-Pakete lokal nur in unserem Unterverzeichnis zu installieren, anstatt global über den Paketmanager der Distribution.

Zunächst muss Python in der Version 3.4 oder neuer installiert werden. Unter Debian Jessie funktioniert das z.B. mit:

# apt-get install python3 python3-venv

(Wobei # bedeutet, dass dieser Befehl mit Superuser-Rechten ausgeführt werden muss.)

Falls ein PDF-Export des erstellten Dokuments erwünscht ist, muss XeTeX installiert sein:

# apt-get install texlive texlive-xetex

Installation der benötigten Python-Module

Wir legen im Verzeichnis env eine virtuelle Python-Umgebung an:

$ python3 -m venv env

Und aktivieren diese:

$ source env/bin/activate

Wir aktualisieren den Python-Paketmanager pip. Ältere Versionen können unter Umständen nicht alle Abhängigkeiten korrekt auflösen. 

$ pip3 install --upgrade pip

Schließlich installieren wir die Python-Module, die wir benutzen werden:

$ pip3 install jupyter numpy matplotlib
